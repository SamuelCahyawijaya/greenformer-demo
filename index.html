Introducing AutoFact
Easily factorize your deep learning model using the low-rank matrix factorization method to enable faster training and inference time on most of model architecture. Use this library to robustly make your models smaller, faster, and cheaper in all runs.

Using the PyAutoFact library is so easy that you might forget it even existed!
