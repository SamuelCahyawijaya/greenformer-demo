from your_project import load_model, train, predict, get_submodules

from py_auto_fact import auto_fact

model = load_model('transformer')

model = auto_fact(module=model, rank=64, solver='random', num_iter=50, submodules=None)

model = train(model)

submodules = get_submodules(model,'bert-base')

model = auto_fact(module=model, rank=64, solver='svd', num_iter=50, submodules=None)

score = predict(model)

More about AutoFact
AutoFact is flexible, easy-to-use, and applicable for multiple scenarios.
Gain up to 2x speed increase while maintaining 93% of the performance by utilising AutoFact.
Performance and efficiency trade-off
Performance and efficiency trade-off of utilizing AutoFact on:
Left: factorization-by-design, Center: post-training factorization, Right: in-context learning factorization
AutoFact performs decomposition to the weight of linear and convolution layers within the models utilizing factorization solvers like random, SVD, or SMNF, thus replacing the linear layers into a Low-rank Encoder-Decoder (LED) layer and the convolution layers into a Convolution Encoder-Decoder (CED) layer. AutoFact supports factorization with either a predefined static rank or dynamic ranks across all layers by computing the rank based on a ratio to the maximum rank of the corresponding layer.
